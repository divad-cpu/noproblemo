# Project Brief

## Product Direction

NoProblemo will be built as a simple, trustworthy web product with a minimal user experience and a careful release path. The long-term requirements remain the source of direction, but this repository currently implements Phase 1 only.

## Phase 1 Objective

Create a stable project foundation that future feature work can build on without exposing secrets or introducing premature product complexity.

## Phase 1 Deliverables

- Verify the Next.js App Router project structure.
- Keep TypeScript, Tailwind CSS, Supabase folder, GitHub, Vercel, and environment variable setup documented.
- Provide safe environment templates.
- Document architecture, database intent, security guardrails, UI direction, roadmap, deployment, and AI readiness.
- Validate the project with linting, type checking, and production build.

## Explicitly Deferred

- Login and authentication
- Supabase database migrations
- Google login
- Apple login
- Payments
- AI features
- Resend email
- Vercel Cron
- `next-intl`
